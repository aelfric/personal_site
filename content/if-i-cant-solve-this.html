----- title: If I Can't Solve This description: Sometimes, software development feels empowering, and sometimes you stare at the same line of code for for hours trying to figure out why it's not working. This was written after one particularly frustrating debugging session. To tune of "If I Can't Love Her" from Beauty and the Beast. tags: stuff_i_wrote, music created_at: 2021-03-07 -----

And in this error state
This bug has sealed my fate
And each attempt I make still hinders progress
And in this office chair
Head banging, and despair
This bug is driving me to utter madness.

Hopeless
As my mind strays
As my eyes glaze
Tracing executions
Helpless
Tired of thinking
Barely blinking
Searching for solutions.

No tests could detect this,
No devtools inspect this,
No pathway I see, if I can’t solve this
No forums could save me
No fixes they gave me
Showed how I could solve this and make my program run
If I can't solve this, I’m done

So much time I’ve watched this screen
Wondering what it all can mean
Flustered and frustrated, I pressed onward

No signs of exceptions
No Future<> rejections
No logs anywhere, to help me solve this
No resource avails me
No help has not failed me
Help for me to solve this, help that could set me free
But it's not to be
If I can't solve this
Let this code be done with me.
